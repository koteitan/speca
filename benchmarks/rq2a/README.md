# RQ2a — RepoAudit C/C++ benchmark harness

This directory holds the evaluation code (`evaluate.py`, `visualize.py`,
`analyze_deep.py`) and ground-truth data (`ground_truth_bugs.yaml`,
`published_baselines.yaml`) for the RQ2 benchmark reported in the paper.

Operator documentation — how to reproduce, what each output file means,
how to extend to new codebases — lives on the documentation site:

→ **[Reproducing RQ2](https://speca.pages.dev/docs/operations/benchmark-rq2a)**
