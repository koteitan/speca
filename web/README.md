# `web/` — Vite + React scaffold (placeholder)

A minimal Vite + React + TypeScript scaffold reserved for a future
browser-based finding viewer. The orchestrator currently exposes its own
JSON event stream consumed by [`speca-cli`](../cli/) (Ink TUI); a
companion web UI is anticipated but not yet implemented here.
