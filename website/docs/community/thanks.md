# Thanks

SPECA は社外の公開データセット・ベンチマーク・ライブラリの上に成り立っています。ここでは **SPECA の外側にあるそれらのリソース** に感謝します。

論文著者やリポジトリのコントリビューターはここでは触れません。論文へのリンクは [リファレンス](../references/paper-fusaka)、開発履歴は [GitHub](https://github.com/NyxFoundation/speca) を見てください。

## ベンチマーク・データセット提供元

- **Sherlock** — Ethereum Fusaka audit contest のデータ
- **Code4rena** — 監査コンテストのアーカイブ
- **CodeHawks** — 監査結果の公開データ
- **RepoAudit** — C/C++ プロジェクトのベンチマーク

これらの公開リソースが無ければ、SPECA の評価とプロパティ抽出はここまで来ていません。

## 依存ツール・ライブラリ

- **Anthropic Claude Code** — 推論エンジン
- **Tree-sitter MCP** — Phase 02c のコード位置解決
- **uv** (Astral) — Python オーケストレータの実行環境
- **Pydantic** — フェーズ間データ契約の検証
- **Docusaurus** — このサイト

## 追記の仕方

漏れている外部リソースがあれば、このファイル (`website/docs/community/thanks.md`) を直接編集する pull request を [NyxFoundation/speca](https://github.com/NyxFoundation/speca) に出してください。
